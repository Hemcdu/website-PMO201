Web Development

Sector: Higher Education

Discipline: Information Technology

Credit Points: 10

Equivalent Units: (CS101)

Prerequisites: NA

Incompatible: NA

Assumed Knowledge: Basic understanding of HTML and CSS

Course Description

This course provides an introduction to web development. Students will learn the fundamentals of creating and maintaining websites, including HTML, CSS, and JavaScript. The course also covers responsive design, accessibility, and best practices for web development. By the end of the course, students will be able to design and develop functional and aesthetically pleasing websites.

Access to highspeed broadband is highly recommended.

Course Learning Outcomes

  1. Understand the structure and functionality of the web and how web browsers render HTML and CSS.
  2. Create and style web pages using HTML and CSS.
  3. Develop interactive websites using JavaScript.
  4. Implement responsive design techniques to ensure websites are accessible on various devices.
  5. Apply best practices in web development to ensure website accessibility and performance.
