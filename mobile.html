Mobile App Development

Sector: Technology

Discipline: Software Engineering

Credit Points: 10

Equivalent Units: (SE401)

Prerequisites: Basic programming knowledge

Incompatible: Web Development Course

Assumed Knowledge: Understanding of software development principles

Course Description

This course focuses on mobile app development for various platforms such as iOS and Android. Students will learn to design and build mobile applications using industry-standard tools and frameworks. Topics covered include user interface design, mobile app architecture, data management, and deployment strategies. By the end of the course, students will be able to develop functional and user-friendly mobile apps.

Access to a computer with necessary software installed is required for practical exercises.

Course Learning Outcomes

  1. Understand mobile app development concepts and platforms.
  2. Design intuitive user interfaces for mobile applications.
  3. Implement mobile app architecture following best practices.
  4. Manage data storage and retrieval in mobile apps.
  5. Deploy mobile applications to app stores and test on real devices.
