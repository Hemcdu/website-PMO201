Cloud Computing

Sector: Technology

Discipline: Information Technology

Credit Points: 10

Equivalent Units: (IT402)

Prerequisites: Basic understanding of networking and virtualization

Incompatible: On-Premises Infrastructure Management Course

Assumed Knowledge: Familiarity with operating systems and web technologies

Course Description

This course introduces students to the fundamentals of cloud computing. It covers various cloud service models (IaaS, PaaS, SaaS), deployment models (public, private, hybrid), and cloud providers (e.g., AWS, Azure). Students will learn about virtualization, containerization, scalability, elasticity, and cost management in the cloud. Practical exercises include deploying applications to the cloud and managing cloud resources.

Access to a computer with internet access is required for hands-on labs.

Course Learning Outcomes

  1. Understand the concepts and principles of cloud computing.
  2. Deploy and manage virtual machines and containers in the cloud.
  3. Implement scalable and resilient cloud architectures.
  4. Utilize cloud services for data storage, computation, and networking.
  5. Optimize costs and performance in cloud environments.
