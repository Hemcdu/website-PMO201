Backend Development

Sector: Higher Education

Discipline: Information Technology

Credit Points: 10

Equivalent Units: (CS303)

Prerequisites: NA

Incompatible: NA

Assumed Knowledge: Basic understanding of web development concepts

Course Description

This course provides an in-depth study of backend web development. Students will learn to create robust, scalable, and secure server-side applications. The course covers server-side programming languages such as Node.js and Python, database management, RESTful APIs, authentication, and deployment techniques. By the end of the course, students will be able to build and maintain backend systems that support frontend applications.

Access to highspeed broadband is highly recommended.

Course Learning Outcomes

  1. Understand server-side programming concepts and languages.
  2. Develop RESTful APIs to facilitate communication between frontend and backend systems.
  3. Manage and interact with databases using SQL and NoSQL technologies.
  4. Implement authentication and authorization mechanisms for secure applications.
  5. Deploy and maintain backend applications on various hosting platforms.
