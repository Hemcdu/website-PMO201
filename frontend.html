Frontend Development

Sector: Higher Education

Discipline: Information Technology

Credit Points: 10

Equivalent Units: (CS202)

Prerequisites: NA

Incompatible: NA

Assumed Knowledge: Basic understanding of HTML, CSS, and JavaScript

Course Description

This course provides an in-depth study of frontend web development. Students will learn to create dynamic, interactive, and responsive web applications using HTML, CSS, and JavaScript. The course also covers modern frontend frameworks and libraries such as React and Vue.js. By the end of the course, students will be able to build advanced user interfaces and enhance the user experience of web applications.

Access to highspeed broadband is highly recommended.

Course Learning Outcomes

  1. Develop advanced HTML and CSS skills to create visually appealing web pages.
  2. Utilize JavaScript to create interactive web applications.
  3. Implement responsive design techniques to ensure web applications are accessible on various devices.
  4. Use modern frontend frameworks and libraries to enhance web development efficiency.
  5. Apply best practices in frontend development to improve performance and user experience.
